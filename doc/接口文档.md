# API 接口文档

## 预约管理接口

### 1. 获取预约列表
- **接口**: `GET /api/reservation`
- **描述**: 获取所有预约记录
- **响应**:
```json
{
  "code": 200,
  "message": "success",
  "data": [...]
}
```

### 2. 获取预约详情
- **接口**: `GET /api/reservation/{id}`
- **描述**: 根据ID获取预约详情

### 3. 创建预约
- **接口**: `POST /api/reservation`
- **请求体**:
```json
{
  "customerName": "张三",
  "phone": "13800138000",
  "reservationTime": "2024-01-01T10:00:00",
  "serviceType": 1,
  "remark": "备注"
}
```

### 4. 更新预约
- **接口**: `PUT /api/reservation/{id}`

### 5. 删除预约
- **接口**: `DELETE /api/reservation/{id}`

### 6. 更新预约状态
- **接口**: `PATCH /api/reservation/{id}/status`
- **请求体**: 状态码 (1=待确认, 2=已确认, 3=已完成, 4=已取消)

## 系统设置接口

### 1. 获取系统配置
- **接口**: `GET /api/setup/config`

### 2. 更新系统配置
- **接口**: `POST /api/setup/config`

### 3. 健康检查
- **接口**: `GET /api/setup/health`

### 4. 用户登录
- **接口**: `POST /ApiService?action=login_login`
- **描述**: 用户登录系统
- **请求体**:
```json
{
  "username": "admin",
  "password": "admin123"
}
```
- **响应成功**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "生成的token",
    "username": "admin"
  }
}
```
- **响应失败**:
```json
{
  "code": 400,
  "message": "用户名或密码错误",
  "data": null
}
```
- **说明**: 默认账号密码为 `admin/admin123`

### 5. 用户退出
- **接口**: `POST /ApiService?action=login_logout`
- **描述**: 用户退出登录
- **请求体**:
```json
{
  "token": "用户的token"
}
```
- **响应成功**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "message": "退出成功"
  }
}
```

### 6. 用户注册
- **接口**: `POST /ApiService?action=login_register`
- **描述**: 新用户注册
- **请求体**:
```json
{
  "username": "newuser",
  "password": "password123"
}
```
- **响应成功**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "message": "注册成功"
  }
}
```
- **响应失败**:
```json
{
  "code": 400,
  "message": "用户名已存在",
  "data": null
}
```
