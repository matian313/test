# API 接口文档

所有接口统一通过 `ApiService` 入口访问，通过 `action` 参数区分不同操作。

## 统一响应格式

所有接口返回格式统一：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

- `code`: 200 表示成功，其他表示失败
- `message`: 提示信息
- `data`: 返回数据（成功时有，失败时为 null）

## 预约管理接口

### 1. 获取预约列表
- **接口**: `GET /ApiService?action=reservation_list`
- **描述**: 获取所有预约记录
- **响应**:
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "customerName": "张三",
      "phone": "13800138001",
      "reservationTime": "2024-01-01T10:00:00",
      "serviceType": 1,
      "status": 1,
      "remark": "备注",
      "createdAt": "2024-01-01T00:00:00",
      "updatedAt": null
    }
  ]
}
```

### 2. 获取预约详情
- **接口**: `GET /ApiService?action=reservation_get&id={id}`
- **描述**: 根据ID获取预约详情
- **参数**:
  - `id`: 预约ID
- **响应**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "customerName": "张三",
    "phone": "13800138001",
    "reservationTime": "2024-01-01T10:00:00",
    "serviceType": 1,
    "status": 1,
    "remark": "备注",
    "createdAt": "2024-01-01T00:00:00",
    "updatedAt": null
  }
}
```

### 3. 创建/更新预约

#### 创建/更新预约（推荐）
- **接口**: `POST /ApiService?action=reservation_save`
- **请求体**:
```json
{
  "customerName": "张三",
  "phone": "13800138001",
  "reservationTime": "2024-01-01T10:00:00",
  "serviceType": 1,
  "remark": "备注"
}
```
- **说明**: 如果不传 `id` 或 `id <= 0` 则为创建，否则为更新

#### 创建预约（兼容旧版）
- **接口**: `POST /ApiService?action=reservation_create`
- **请求体**:
```json
{
  "customerName": "张三",
  "phone": "13800138001",
  "reservationTime": "2024-01-01T10:00:00",
  "serviceType": 1,
  "remark": "备注"
}
```

#### 更新预约（兼容旧版）
- **接口**: `POST /ApiService?action=reservation_update`
- **请求体**:
```json
{
  "id": 1,
  "customerName": "张三",
  "phone": "13800138001",
  "reservationTime": "2024-01-01T10:00:00",
  "serviceType": 1,
  "remark": "备注"
}
```

#### 字段说明
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `id` | int | 否 | 预约ID（创建时不传，更新时必填） |
| `customerName` | string | 是 | 客户姓名 |
| `phone` | string | 是 | 联系电话 |
| `reservationTime` | string | 是 | 预约时间 (ISO格式) |
| `serviceType` | int | 否 | 服务类型（1=咨询服务, 2=保养服务, 3=维修服务, 4=其他服务） |
| `remark` | string | 否 | 备注 |

### 4. 删除预约
- **接口**: `GET /ApiService?action=reservation_delete&id={id}`
- **描述**: 根据ID删除预约
- **参数**:
  - `id`: 预约ID

### 5. 更新预约状态
- **接口**: `POST /ApiService?action=reservation_updatestatus`
- **描述**: 更新预约状态
- **请求体**:
```json
{
  "id": 1,
  "status": 2
}
```
- **状态说明**:
  - `1`: 待确认
  - `2`: 已确认
  - `3`: 已完成
  - `4`: 已取消

## 系统设置接口

### 1. 获取系统配置
- **接口**: `GET /ApiService?action=setup_getconfig`
- **描述**: 获取系统配置
- **响应**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "systemName": "AI预约服务系统",
    "version": "1.0.0",
    "isMaintenance": false
  }
}
```

### 2. 更新系统配置
- **接口**: `POST /ApiService?action=setup_updateconfig`
- **请求体**:
```json
{
  "systemName": "AI预约服务系统",
  "version": "1.0.0",
  "isMaintenance": false
}
```

### 3. 健康检查
- **接口**: `GET /ApiService?action=setup_healthcheck`
- **描述**: 检查系统运行状态
- **响应**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "status": "OK",
    "time": "2024-01-01T00:00:00"
  }
}
```

## 登录注册接口

### 1. 用户登录
- **接口**: `POST /ApiService?action=login_login`
- **描述**: 用户登录系统
- **请求体**:
```json
{
  "username": "admin",
  "password": "admin123"
}
```
- **响应成功**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "生成的token",
    "username": "admin"
  }
}
```
- **响应失败**:
```json
{
  "code": 400,
  "message": "用户名或密码错误",
  "data": null
}
```
- **说明**: 默认账号密码为 `admin/admin123`

### 2. 用户退出
- **接口**: `POST /ApiService?action=login_logout`
- **描述**: 用户退出登录
- **请求体**:
```json
{
  "token": "用户的token"
}
```
- **响应成功**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "message": "退出成功"
  }
}
```

### 3. 用户注册
- **接口**: `POST /ApiService?action=login_register`
- **描述**: 新用户注册
- **请求体**:
```json
{
  "username": "newuser",
  "password": "password123"
}
```
- **响应成功**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "message": "注册成功"
  }
}
```
- **响应失败**:
```json
{
  "code": 400,
  "message": "用户名已存在",
  "data": null
}
```
